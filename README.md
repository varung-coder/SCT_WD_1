# Luminar : A modern Responsive Landing Page

## Overview
Welcome to the **SCT_WD_1** Responsive Landing Page! This project is a premium, modern, and highly interactive landing page built for a creative digital agency named **Luminar**. It is designed to be fully responsive across desktop, tablet, and mobile devices. 

The website highlights a professional aesthetic with features such as glassmorphism, gradient text animations, scroll-reveal transitions, and dynamic hover effects.

## 🔗 Live Demo
Experience the live website here: https://varung-coder.github.io/SCT_WD_1/

## 🌟 Key Features
- **Interactive Glassmorphism Navbar:** A transparent sticky header that elegantly blurs into a frosted-glass effect as you scroll down the page.
- **Dynamic Typing Effect:** An engaging typewriter animation in the hero section that cycles through key concepts.
- **Scroll Reveal Animations:** Content gracefully fades and slides into view as it enters the viewport using the `IntersectionObserver` API.
- **Premium UI Elements:** Features soft shadows, glowing card hover effects, ripple button clicks, and gradient text animations.
- **Fully Responsive Design:** A mobile-first approach ensuring perfect alignment and a mobile-friendly hamburger menu.
- **Scroll Progress Indicator:** A custom gradient progress bar fixed to the top of the screen tracking your scroll depth.
- **Back-to-Top Button:** A smooth-scrolling floating button that appears after scrolling down.

## 🛠️ Tech Stack
This project was built purely with core web technologies, without the use of any heavy frameworks:
- **HTML5:** Semantic and accessible document structure.
- **CSS3:** Flexbox, CSS Grid, custom properties (variables), keyframe animations, and modern aesthetics like `backdrop-filter`.
- **Vanilla JavaScript:** DOM manipulation, event listeners, and scroll tracking for interactive UI elements.
- **Fonts & Icons:** Google Fonts (*Space Grotesk* and *Inter*) and FontAwesome for iconography.

## 🎨 Design Highlights
The design takes inspiration from high-end SaaS platforms and digital agencies. By relying on a "Deep Slate" dark mode palette (`#020617`), vibrant neon accents (Purples, Blues, and Roses), and generous whitespace, the website achieves a look that is both cutting-edge and highly professional.
